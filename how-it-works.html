How it works

Record sessions → detect trends → adjust the plan. The goal is simple: more wins, fewer “why isn’t this working?” moments.

1️⃣

Log what happened

After a session, you note the skill, duration, success rate, and the reward you used (treat, toy, praise).

2️⃣

Compawnion analyses patterns

It looks for repeat signals: motivation dips, error types, distraction sensitivity, and plateau warnings.

3️⃣

Daily plan updates

Your plan changes day-by-day: shorter reps, different reward schedules, or a step back to rebuild confidence.

4️⃣

Clear “do this next” guidance

You get concrete instructions: number of reps, when to reward, what to ignore, and what to end on.

What data is recorded?

You can keep it super light. The more you track, the smarter it gets — but it’s not all-or-nothing.

  • Skill / exercise name
  • Session length (e.g., 3–5 mins)
  • Success rate (rough %)
  • Reward type (treat / toy / praise)
  • Reward value (low / medium / high)
  • Context (indoors / outdoors / distractions)
  • Notes (what went wrong, what worked)

Adaptive examples

Compawnion might recommend:

  • Cut treats when reliability is high — replace with praise/play + variable schedule
  • Increase treats when confidence drops — higher value, shorter reps, more wins
  • Switch to toy motivation if food distracts or the dog prefers play
  • Change toy type based on response: tug vs chase vs squeak vs fetch
  • Reduce difficulty (distance/distraction/duration) when plateau signals appear
